# Dados

Contém todos os dados utilizados no projeto. **Os arquivos de dados não são versionados no Git.**

## Estrutura

- `raw/` - Dados brutos (IoT, satélite, meteorológicos)
- `interim/` - Dados intermediários após pré-processamento
- `processed/` - Dados finais prontos para modelagem
- `external/` - Dados de fontes externas (shapefiles, etc.)
