# Hardware

Contém toda a documentação relacionada ao hardware desenvolvido para o sistema de coleta de dados em campo.

## Estrutura

- `schematics/` - Esquemas elétricos e diagramas de conexão
- `firmware/` - Código embarcado para os microcontroladores (ESP32 + LoRaWAN)
- `datasheets/` - Folhas de dados dos componentes utilizados
